- リニューアル対象: https://ehm-saijo-ah.esnet.ed.jp/

- 成果物: トップページモックアップ `index.html`(単一ファイル / レスポンシブ / JSアニメーション込み)

## 1. UI/UX分析 — 参考サイトが「モダン・カッコいい・スタイリッシュ」に見える理由

藍野学院系サイトに代表される近年のハイエンド学校サイトは、以下の要素の組み合わせで
「学校サイトらしからぬ」洗練された印象を作っている。

### 1-1. 配色 — 「1ブランドカラー + 大量の白」
- 彩度の高い色を多用せず、**ブランドカラー1色 + 白(オフホワイト)+ 墨色テキスト**にほぼ限定。
- 色数を絞ることで写真が主役になり、ブランドカラーがアクセントとして強く記憶に残る。
- 背景に純白 (#fff) ではなくわずかにトーンを落としたオフホワイトを使い、上品な質感を出す。

### 1-2. 余白 — セクション間を「空けすぎなくらい」空ける
- セクション上下に 100〜160px 級の余白。情報密度を意図的に下げ、1画面1メッセージにする。
- 従来型の学校サイト(情報を詰め込む)との最大の差はここ。余白そのものが「洗練」の記号。

### 1-3. タイポグラフィ — 日本語見出し × 英語ディスプレイの二層構造
- 見出しは「大きな英語(NEWS / COURSE / SCHOOL LIFE)+ 小さな日本語ラベル」の組み合わせ。
  英語は装飾、日本語が機能、という役割分担で可読性とスタイルを両立。
- 日本語はモダンゴシック(Noto Sans JP / Zen Kaku Gothic 系)を太めのウェイトで、
  英語はジオメトリックサンセリフを letter-spacing 広めで使う。
- 巨大な英字を背景に薄く敷く「ゴーストテキスト」も定番の演出。

### 1-4. 画像の魅せ方 — 生徒の「顔と手元」を大きく
- 校舎の引き写真ではなく、**生徒の表情・実習の手元のクローズアップ**を全幅で使う。
- 角丸・円形・斜めのマスク、画像のはみ出し(ブリード)配置で静的なグリッドを崩す。
- ヒーローはファーストビュー全面(100svh)の写真/動画 + 短いコピー1行。

### 1-5. スクロール体験 — 「動きの節度」
- スクロールに応じたフェードイン/スライドイン(IntersectionObserver)。
- 動きは 0.6〜0.8s のイージング1種類に統一し、派手にしすぎない。統一感=高級感。
- 固定ヘッダーはスクロールで縮小・背景色付与。ページ内の現在地を常に見失わせない。
- 数字のカウントアップ、無限マーキー(帯状に流れる写真/文字)など「1〜2個だけ」目玉の動きを入れる。

### 1-6. ナビゲーション — 迷わせないための単純化
- グローバルナビは 6±1 項目に厳選し、残りはフッターの完全サイトマップへ。
- 「受験生へ」「資料請求」など主要CTAはヘッダー右端に色付きボタンで常設。
- モバイルはフルスクリーンのハンバーガーメニュー(開閉アニメーション付き)。

## 2. コンセプト設計 — 西条農業高校 × モダンスタイル

### 2-1. デザインコンセプト

> **「耕す、未来を。」 — AGRI × TECH × SAIJO**

西条市は石鎚山の伏流水「うちぬき」が湧く"水の都"であり、道前平野の農業地帯。
「土と水の学校」というローカルな確かさと、スマート農業・食品科学という先進性を、
**深い緑(森・土)+ 湧き水の白 + 実りの金**の3色で表現する。

- 信頼感・安心感: 深緑を基調にした落ち着いた配色、整ったグリッド、読みやすい本文
- 先進性: 大胆な英字タイポ、全画面ヒーロー、スクロール演出、データ(数字)の見せ方

### 2-2. カラースキーム(WCAG コントラスト配慮)

| 役割 | 色 | 用途 |
|---|---|---|
| Primary | `#1B4332`(ディープフォレスト) | 見出し・フッター・主要ボタン。白文字とのコントラスト比 ≈ 10.9:1 |
| Primary Light | `#2D6A4F` | ホバー・サブ要素。白文字と ≈ 6.6:1 |
| Accent | `#E0A422`(ハーベストゴールド) | CTA・強調。**墨色文字**と組み合わせて ≈ 8:1 を確保 |
| Base | `#F7F5F0`(ライムストーン=うちぬきの白) | 背景 |
| Ink | `#1F2421`(墨色) | 本文。Base 背景と ≈ 15:1 |
| Sub Ink | `#5C6660` | 補足テキスト。Base 背景と ≈ 5.3:1(AA) |

※ 金色背景に白文字は使わない(コントラスト不足)。金は「墨色文字の下地」または細部の装飾に限定。

### 2-3. タイポグラフィ

- 日本語: **Zen Kaku Gothic New**(fallback: Noto Sans JP, system-ui)
  — 骨格がモダンで、公立校の「まじめさ」を損なわない
- 英語ディスプレイ: **Outfit**(fallback: Montserrat, sans-serif)
  — セクション見出しの大文字英字、数字のカウンター用
- 本文 16px / 行間 1.9。見出しは clamp() で流体スケール。

### 2-4. 西農らしさの翻訳ルール

| 西農の資産 | モダンスタイルへの翻訳 |
|---|---|
| 実習(栽培・食品加工・測量など) | 手元・表情のクローズアップ写真を全幅/ブリード配置 |
| 3つの学科の専門性 | 学科ごとのテーマカラーを持つ大型カード + ホバー演出 |
| 農業クラブ・資格・地域連携の実績 | 「数字で見る西農」カウントアップセクション |
| 石鎚山・うちぬき・道前平野 | 配色(緑×白×金)と、等高線/水紋モチーフの背景パターン |
| 120年超の歴史 | ヒーローの「SINCE 1900」表記、フッターの校訓 |

## 3. ワイヤーフレーム(トップページ構成案)

| # | セクション | 役割 |
|---|---|---|
| 0 | 固定ヘッダー | ロゴ+主要ナビ6項目+CTA「中学生の皆さんへ」。スクロールで縮小。モバイルは全画面メニュー |
| 1 | ヒーロー(100svh) | 実習風景の全面写真+コピー「耕す、未来を。」+ SINCE 1900。第一印象で先進性を提示 |
| 2 | 緊急・新着バー | 警報時の臨時休業等を最上部で確実に告知(公式サイトとしての安心感) |
| 3 | NEWS | 新着情報をカテゴリタグ付きリストで。既存サイトの更新文化をそのまま受け継ぐ |
| 4 | ABOUT + 数字で見る西農 | 学校の立ち位置を一言で+実績をカウントアップ数字で(信頼と先進性の両立) |
| 5 | COURSE 学科紹介 | 3学科の大型カード。各学科ページへの最重要導線 |
| 6 | FEATURES 西農の魅力 | 実習・資格・地域連携を交互レイアウトで深掘り |
| 7 | SCHOOL LIFE | 年間行事・日常のギャラリーをマーキー(流れる帯)で感情的に見せる |
| 8 | 進路 CAREER | 進学・就職の実績と主な進路先。保護者向けの安心材料 |
| 9 | CTA バナー | 「中学生の皆さんへ」学校見学・体験入学への強い導線 |
| 10 | フッター | 完全サイトマップ(学校案内/学科/進路/部活動/事務室/アクセス)+住所・連絡先 |

既存コンテンツ(学校案内・行事予定・進路状況・部活動・事務室・警報時の対応 等)への導線は
ヘッダー6項目+フッターサイトマップで全て維持する。

## 4. 実装メモ

- `index.html` 単一ファイル。CSSはカスタムプロパティベースの手書き(Tailind不使用で依存ゼロ)、
  Google Fonts のみ外部読み込み(オフライン時はシステムフォントにフォールバック)。
- 写真素材は未支給のため、CSSグラデーション+インラインSVGパターンのプレースホルダーで構成。
  `<figure data-photo="...">` に実写真を `background-image` で差すだけで置換可能。
- JS: IntersectionObserver によるフェードイン/カウントアップ、ヘッダー縮小、
  ハンバーガーメニュー、マーキー。`prefers-reduced-motion` 対応済み。
- アクセシビリティ: スキップリンク、ランドマーク、aria-label、フォーカスリング、
  コントラスト比 AA 以上(本文・ボタンは AAA 近傍)。
