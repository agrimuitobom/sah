# 西条農業高校 Webサイトリニューアル モックアップ

愛媛県立西条農業高等学校の公式サイトを「モダン・カッコいい・スタイリッシュ」に
リニューアルするための企画設計とトップページのモックアップです。

- デザイン参考: 藍野学院サイト(https://ainogakuen.ed.jp/)のテイスト
- 現行サイト: https://ehm-saijo-ah.esnet.ed.jp/

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページのモックアップ(単一ファイル・依存はGoogle Fontsのみ) |
| `docs/design-concept.md` | UI/UX分析・デザインコンセプト・ワイヤーフレームの設計書 |

## 使い方

`index.html` をブラウザで開くだけで動作します。

写真素材は CSS グラデーションのプレースホルダーです。
`.ph` 要素に `background-image` を指定するだけで実写真に差し替えられます。

```css
.ph-agri{ background-image:url("images/farm.jpg"); }
```

学科名・数字実績・住所などのテキストはモック用の仮データです
(該当箇所に「※要調整」の注記あり)。公開前に実データへの差し替えが必要です。
